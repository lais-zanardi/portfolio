Plataforma de Análise do Comércio Exterior Paulista

Projeto acadêmico foi desenvolvido em equipe, conta com uma plataforma web interativa que analisa e visualiza o desempenho dos municípios do estado de São Paulo no comércio exterior, utilizando dados abertos do Ministério do Desenvolvimento, Indústria, Comércio e Serviços.

Minha principal contribuição foi a prototipagem do site e desenvolvimento de gráficos interativos para análise de dados de comércio exterior.
